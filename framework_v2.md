# 烟花视频画面分析框架 v2.0

## 目标
从烟花视频中提取**产品性能数据**（Layer 0），并在此基础上派生**市场分析指标**（Layer 1）。
提取分为两路：**视频中有参数文字 → OCR抓取** / **纯燃放画面 → AI看图拆解**。

---

## Layer 0：产品性能原始数据提取

### 路径A：视频中出现参数文字 → OCR直接抓取
触发条件：视频任意帧检测到含参数的文字区域（包装标签、字幕说明、标题卡）。

| 抓取字段 | 示例 |
|---------|------|
| 品名 / 品牌 / 型号 | "Red Apple Fireworks — High Falutin" |
| 发数 | "100 shot"、"100发" |
| 筒径 / 规格 | "1.75寸"、"500g cake"、"200g cake"、"30mm" |
| 药量 | "60g"、"500g" |
| 燃放时长 | "Duration: 60s" |
| 效果名称 | "Peony"、"牡丹"、"Brocade" |

### 路径B：纯燃放画面无文字 → AI看图拆解

#### B1. 实际燃放时长
| 原子指标 | 测量方式 |
|---------|---------|
| 燃放开始帧 | 第一发炸点出现（画面亮度突变 + 运动矢量激增） |
| 燃放结束帧 | 最后一发消散（连续60帧无新炸点） |
| 实际燃放时长 | 结束帧 - 开始帧，精确到0.1秒 |

#### B2. 实际发数
| 原子指标 | 测量方式 |
|---------|---------|
| 总发射次数 | 逐帧炸点检测 + 相邻帧同一炸点去重 |
| 哑弹/空管数 | 发射间隔 > 3倍平均间隔的空档，标记可疑空位 |
| 有效发数 | 总发射 - 哑弹数 |

#### B3. 筒子规格（视觉推断）
- 发射位数量：统计画面中可辨识的独立发射口/管口数量
- 排列方式：单排 / 双排 / 多排 / 矩阵 / V型（通过发射位空间分布聚类判定）
- 管间距：相邻发射口画面像素距离
- 筒径估计：炸点起始光斑大小 + 发射口间距 → 查经验表推算口径范围

#### B4. 药量（关键词优先 + 视觉推断兜底）
优先级：
1. **关键词直接命中**（500g / 200g / 60g / 350g / 120g / 30g）→ 直接填入对应数字
2. OCR提取 → 直接填入
3. 视觉推断（筒径 + 炸点直径 + 效果类型 + 时长查对照表）→ 标注"视觉推算值" + 置信度

---

### 节奏结构逐排拆解

#### 分段触发条件（满足任一即切新段）
1. 连续两发间隔 > 该视频平均间隔 × 2.5
2. 发射模式变化（齐发 ↔ 连发切换）
3. 炸点空间位置突变（偏移 > 画面宽度 20%）
4. 效果类型明显切换
5. 颜色主题明显切换

#### 发射模式穷举分类

| 模式 | 判定规则 |
|------|---------|
| **齐上** | 同一帧或相邻3帧内 ≥3发同时出现，空间分散 |
| **齐放** | 同一帧或相邻2帧内 ≥2发同时出现，空间集中 |
| **快连发** | 连续发射间隔 <0.3秒，持续 ≥5发 |
| **连发** | 连续发射间隔 0.3~1.5秒，逐个发射 |
| **慢连发** | 连续发射间隔 >1.5秒，逐个发射 |
| **V型** | 炸点空间轨迹呈V字形左右对称外扩 |
| **直上** | 轨迹近似垂直，水平位移 <画面宽度10% |
| **扇形** | 炸点从中心向外呈扇形扩散 |
| **交叉** | 左右两侧交替发射 |
| **W型** | 炸点轨迹呈W或锯齿形 |
| **Z型** | 炸点轨迹呈Z字形 |
| **单发** | 段内仅1发 |

#### 每段标注项

| 段序号 | 开始时间 | 结束时间 | 发射数 | 时长 | 发射模式 | 效果 | 颜色 |
|--------|---------|---------|--------|------|---------|------|------|

#### 节奏类型判定
由各段发射频率曲线自动分类：
- **渐进式**：发射频率逐步递增
- **脉冲式**：高低频率交替
- **均匀式**：全段频率波动 <20%
- **高潮前置式**：前30%时间集中 >60%发射

---

### 效果识别

#### 识别方式
用户提供**效果清单**（效果名称 + 关键帧参考图），AI将视频中观察到的每发效果与参考图做视觉相似度匹配，取最接近的标签。

#### 每发标注
| 指标 | 说明 |
|------|------|
| 效果名称 | 匹配清单中最接近的效果 |
| 匹配置信度 | 高 / 中 / 低 |
| 该效果持续帧数 | 从出现到完全消散 |
| 颜色 | 该发的颜色标签 |

#### 效果序列格式
```
[0.0s] 牡丹(红)×3 → [3.2s] 椰树(金)×5 → [5.0s] 菊花(绿)×4 → ...
```

---

### 花色识别

| 指标 | 测量方式 |
|------|---------|
| 每发主色 | 炸点区域RGB均值 → 匹配13色标准色板 |
| 颜色变化 | 单发内色相偏移检测（不同帧颜色标签是否变化） |
| 颜色组合 | 同一帧多发的颜色组合标记 |
| 花束形态 | 匹配效果清单中的形态分类 |
| 炸点直径(相对) | 炸点外接圆直径 / 画面短边 |
| 尾迹长度(相对) | 尾迹像素长度 / 画面短边 |
| 是否有内芯 | 炸点中心是否有明显更亮的区域（0/1） |

**13色标准色板（预定义RGB范围）：**
`红 橙 黄 金 绿 青 蓝 紫 粉 白 银 玫红 柠檬黄`

## Layer 1：市场分析派生指标（5大分析模块）

所有 Layer 1 指标均从 Layer 0 原始数据 + 平台公开数据（播放/点赞/评论）派生计算，零新增人工采集。

---

### 模块1：产品竞争力分析 (Product Competitiveness)
> 核心问题：这个产品本身好不好？和竞品比处于什么水平？

#### 1.1 规格达标度
| 指标 | 计算方式 |
|------|---------|
| 发数偏差率 | abs(实际发数 - 标称发数) / 标称发数 × 100% |
| 时长偏差率 | abs(实际时长 - 标称时长) / 标称时长 × 100% |
| 哑弹率 | 哑弹数 / 总发射数 × 100% |
| 规格一致性 | OCR/关键词参数与视觉实测的吻合度评分 |

#### 1.2 性能表现
| 指标 | 计算方式 |
|------|---------|
| 发射密度 | 有效发数 / 实际时长（发/秒） |
| 效果种类数 | 去重效果名称计数 |
| 色彩种类数 | 去重颜色计数 |
| 效果切换频率 | 效果变化次数 / 总时长 |
| 炸点高度一致性 | 各发炸点Y坐标的变异系数(CV) |
| 节奏复杂度 | 段数 × 发射模式种类数 |
| Finale力度 | Finale段发射数 / 总发射数 × Finale段内发射频率 |

#### 1.3 同品类定位
| 指标 | 计算方式 |
|------|---------|
| 规格竞争力指数 | (发数 × 效果种类) / (时长 × 哑弹率+1) |
| 同品类百分位排名 | 上述指数在同类目（cake/fountain）中的排名 |
| 差异化程度 | 该产品效果组合在同品类中的稀缺度（出现频次越低越稀缺） |

---

### 模块2：供应链价值分析 (Supply Chain Value)
> 核心问题：这个产品适合做贸易吗？合规吗？利润空间如何？

#### 2.1 合规性
| 指标 | 计算方式 |
|------|---------|
| 药量合规标记 | 药量是否在目标国法规上限内（如美国CPSC 500g上限） |
| 筒径合规标记 | 筒径是否符合目标国1.4G/1.3G分类标准 |
| 时长合规标记 | 燃放时长是否在目标国上下限内 |
| 标签语言 | OCR检测到的语言种类（与目标市场语言是否匹配） |

#### 2.2 品类竞争力
| 指标 | 计算方式 |
|------|---------|
| 品类内综合评分 | 规格竞争力 × 效果丰富度 × 合规系数（0/1） |
| 价格带定位 | 基于发数+筒径+时长+效果种类的回归推算 → low/mid/high/premium |
| 同价格带最佳选择 | 同价格带内综合评分最高的产品 |
| 最小发射间隔 | 所有相邻发射间隔的最小值（安全阈值参考：<0.1秒为高风险） |

#### 2.3 供给端信号
| 指标 | 计算方式 |
|------|---------|
| 品牌/厂商识别 | OCR提取的品牌名 |
| 产品线覆盖度 | 同品牌在采集库中的产品条目数 |
| 代工可能性标记 | 包装语言与UP主所在地区不一致（如中文包装+美国UP主） |

---

### 模块3：消费者需求分析 (Consumer Demand)
> 核心问题：消费者喜欢什么？什么驱动购买？

#### 3.1 体验质量
| 指标 | 计算方式 |
|------|---------|
| 效果丰富度总分 | 色彩种类 + 形态种类 + 特殊效果种类 |
| Wow Moment密度 | Wow Moment次数 / 总时长（次/秒） |
| Wow Moment定义 | 同时满足：炸点直径>画面短边30% + 多色 + 区域亮度峰值 |
| 节奏刺激性 | 发射频率峰值（发/秒） |
| 烟雾干扰度 | 后半段画面被烟雾遮挡的时长占比（负面影响） |

#### 3.2 购买意向信号（从平台公开数据）
| 指标 | 计算方式 |
|------|---------|
| 综合互动率 | (点赞 + 评论) / 播放量 |
| 评论情感倾向 | 评论中正面/负面关键词比例（需评论数据） |
| 购买意图关键词 | 评论命中"buy""price""where to get""shipping"的次数 |
| 复购信号 | 评论命中"buy again""restock""another one"的次数 |
| 问询频率 | 评论中问价格/渠道/购买途径的比例 |

#### 3.3 偏好趋势
| 指标 | 计算方式 |
|------|---------|
| 同效果偏好热度 | 某效果在不同产品中出现的频率变化（周/月） |
| 同颜色偏好热度 | 某颜色在不同产品中出现的频率变化 |
| 节奏类型偏好 | 各节奏类型的平均互动率对比 |
| 价格带关注度 | 各价格带视频的平均播放量（哪个价格带被看最多） |

---

### 模块4：内容传播力分析 (Content Virality)
> 核心问题：这个产品的视频内容传播效率如何？

#### 4.1 内容质量
| 指标 | 计算方式 |
|------|---------|
| 内容信息密度 | 总发射数 / 燃放段视频时长（发/秒） |
| 效果多样性 | 去重效果种类 + 颜色种类 |
| 节奏可观赏性 | 各段时长标准差（变化越大视觉节奏越丰富） |
| Finale冲击力 | Finale段发射密度 × Finale段效果种类数 |

#### 4.2 传播效率（从平台公开数据）
| 指标 | 计算方式 |
|------|---------|
| 互动转化率 | (点赞 + 评论) / 播放量 |
| 播放量等级 | 千级 / 万级 / 十万级 / 百万级 |
| 爆款潜力指数 | 效果丰富度 × Finale力度 × 互动转化率 |
| 长尾衰减率 | 发布后第2周播放增量 / 第1周播放增量 |

#### 4.3 内容差异化
| 指标 | 计算方式 |
|------|---------|
| 效果组合稀缺度 | 该效果组合在采集库中的出现频率（越低越稀缺） |
| 品类内容饱和度 | 同品类视频总数 |
| 差异化指数 | 稀缺度 / 内容饱和度（越高越容易脱颖而出） |

---

### 模块5：市场趋势分析 (Market Trends)
> 核心问题：这个品类在涨还是跌？什么在变热？

#### 5.1 品类热度
| 指标 | 计算方式 |
|------|---------|
| 品类视频发布量趋势 | 同类产品近期视频数量的同比/环比变化 |
| 品类平均互动率趋势 | 同类产品互动率的移动平均 |
| 品类新进入者数量 | 新出现的品牌/UP主频道数量 |
| 季节性指数 | 各月发布量占全年比例（识别销售旺季） |

#### 5.2 产品演化
| 指标 | 计算方式 |
|------|---------|
| 效果组合变化 | 新效果组合首次出现的时间（标记创新方向） |
| 规格趋势 | 发数/时长的均值变化方向 |
| 价格带迁移 | 各价格带产品占比变化（消费升级/降级信号） |
| 安全标准化趋势 | 合规产品占新增产品的比例变化 |

#### 5.3 区域差异
| 指标 | 计算方式 |
|------|---------|
| UP主地域分布 | 频道地区标签统计 → 市场热度地图 |
| 各区域偏好效果 | 不同地区视频中效果种类Top差异 |
| 各区域价格带差异 | 不同地区热门产品价格带分布 |

---

## 明确不采集的指标
- 拍摄分辨率、画面抖动、机位数量
- UP主面部、人物数量、表情
- 引线可见性、点火方式
- 包装外观美学（除非含参数文字，用于OCR）
- 背景环境细节、安全距离

## 附录A：效果清单标准格式

### 字段定义
| 字段 | 类型 | 说明 |
|------|------|------|
| 效果名称 | string | 中文名 / 英文名 |
| 炸点形态 | enum | 球形 / 柱状 / 扇形 / 环形 / 不规则 / 无炸点(地面喷) |
| 对称性 | enum | 径向对称 / 左右对称 / 不对称 / 不适用 |
| 尾迹特征 | enum | 无尾迹 / 短尾迹(<炸点半径) / 长尾迹(>炸点半径) / 下垂尾迹 / 闪烁尾迹 |
| 炸点直径(相对) | enum | 小(<15%画面短边) / 中(15-30%) / 大(30-50%) / 超大(>50%) |
| 持续时间(典型) | range | 从炸开到完全消散的典型时长范围（秒） |
| 颜色特征 | string | 典型颜色组合 |
| 上升特征 | enum | 有可见上升管 / 无上升管 / 地面直接喷 |
| 特殊标记 | string | 哨音 / 噼啪声 / 变色 / 炸中炸 / 旋转等附加特征 |
| 易混淆效果 | string | 与哪些效果容易混淆，区别在哪里 |
| 参考帧 | image | 关键帧截图（最优先的匹配依据） |

### 示例条目

**示例1：牡丹 (Peony)**
```
效果名称: 牡丹 / Peony
炸点形态: 球形
对称性: 径向对称
尾迹特征: 无尾迹或极短尾迹
炸点直径(相对): 中到大 (15-40%)
持续时间(典型): 2-5秒
颜色特征: 红/金/绿/紫/白/多色组合，单发可单色或多色
上升特征: 有可见上升管（但花束本体无尾迹）
特殊标记: 无
易混淆效果: 菊花 — 区别是菊花有长拖尾星光，牡丹光点为圆形光斑无尾迹
```

**示例2：椰树 (Palm Tree)**
```
效果名称: 椰树 / Palm Tree
炸点形态: 球形(顶部) + 弧形下垂
对称性: 径向对称
尾迹特征: 长下垂尾迹（光点从顶部弧形下落）
炸点直径(相对): 中到大 (15-35%，含下垂范围更大)
持续时间(典型): 4-8秒（含上升+炸开+下垂全程）
颜色特征: 多为金色/银色，顶部炸点常有闪点
上升特征: 有粗上升管（形成"树干"）
特殊标记: 光点必须有"先上升→顶部炸开→弧形下落"的完整抛物线轨迹
易混淆效果: 柳条 — 区别是椰树有明确的顶部球形炸开点，柳条无炸点、直接线性下垂
```

**示例3：喷泉 (Fountain)**
```
效果名称: 喷泉 / Fountain / Ground Fountain
炸点形态: 柱状或扇形喷射（无升空、无空中炸点）
对称性: 多为左右对称
尾迹特征: 无尾迹（火花从管口直接喷出）
炸点直径(相对): 高度类（不高过画面中线的地面级效果）
持续时间(典型): 15-60秒
颜色特征: 银白/金/红/绿，常有变色
上升特征: 无上升管，地面直接喷
特殊标记: 大量烟雾，可能有噼啪声(crackle)
易混淆效果: 无
```

**示例4：菊花 (Chrysanthemum)**
```
效果名称: 菊花 / Chrysanthemum
炸点形态: 球形
对称性: 径向对称
尾迹特征: 长拖尾（每条光点带长尾，尾迹明显长于炸点半径）
炸点直径(相对): 中到大 (20-40%，但含尾迹时视觉范围更大)
持续时间(典型): 3-6秒
颜色特征: 金/银/白/紫/绿
上升特征: 有可见上升管
特殊标记: 尾迹末端可能带闪点
易混淆效果: 牡丹 — 区别是菊花有明显长尾迹星光，牡丹光点为圆形光斑无尾迹
```
